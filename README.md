# Reinforcement Learning Tutorial: DQN on CartPole

This repository contains a comprehensive tutorial on Reinforcement Learning (RL), specifically implementing a Deep Q-Network (DQN) to solve the classic CartPole problem using PyTorch and Gymnasium.

## 🚀 One-Click to Kaggle

You can run this notebook directly on Kaggle by clicking the button below:

[![Kaggle](https://kaggle.com/static/images/open-in-kaggle.svg)](https://kaggle.com/kernels/welcome?src=https://github.com/greathousesh/reinforcelearning/blob/main/rl_tutorial.ipynb)

## 📋 Content
- **Introduction to RL**: Basic concepts and terminology.
- **Environment Setup**: Using Gymnasium for simulation.
- **Deep Q-Network**: Architecture and theory.
- **Experience Replay**: Implementation of a Replay Buffer.
- **Training Loop**: Step-by-step training process and visualization.

## 🛠️ Local Setup
If you want to run it locally, make sure you have the following installed:
```bash
pip install gymnasium[classic_control] torch matplotlib numpy
```
